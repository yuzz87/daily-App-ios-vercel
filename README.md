# Daily Life App

日常生活の管理をまとめて行える PWA（Progressive Web App）です。
スマートフォンのホーム画面に追加することで、ネイティブアプリに近い感覚で利用できるように心がけました。

---

## 概要

カレンダー・タイマー・コーヒー記録・ボイスメモを 1 つのアプリに統合した個人向けライフログツールです。
登録したユーザーのみ利用できる認証機能を備えています。

## 主な機能

### 1. カレンダー（予定管理）

月・週・日・年・スケジュールの 5 つのビューを切り替えて予定を管理できます。

**できること:**

- 予定の作成・編集・削除（モーダル UI）
- 祝日の自動表示
- 予定へのカラーラベル付け
- キーワードによる予定の検索・絞り込み
- サイドバーにミニカレンダーと直近の予定一覧を表示

**ビュー一覧:**

| ビュー             | 説明                              |
| ------------------ | --------------------------------- |
| 月ビュー           | カレンダー形式で 1 ヶ月を一覧表示 |
| 週ビュー           | 1 週間を縦軸（時間）で表示        |
| 日ビュー           | 1 日の予定を時間軸で表示          |
| 年ビュー           | 12 ヶ月を一覧表示                 |
| スケジュールビュー | 予定をリスト形式で表示            |

---

### 2. コーヒー豆管理（AI 画像解析）

PostCoffee のパッケージ画像を撮影・アップロードするだけで、コーヒー豆の情報を自動で読み取り記録できます。

**登録フロー:**

```
1. パッケージ画像を選択（スマホはカメラ撮影、PC はファイル選択）
      ↓
2. AI（Tesseract OCR + 独自解析）がパッケージのテキストを読み取り
      ↓
3. ブランド・産地・焙煎度・フレーバーノーツ等を自動抽出
      ↓
4. 確認・編集画面で内容を修正して保存
      ↓
5. コーヒー豆一覧に追加される
```

**記録できる情報:**

| フィールド             | 説明                       |
| ---------------------- | -------------------------- |
| ブランド・商品コード   | パッケージ上部から読み取り |
| 商品名（英語・日本語） | OCR + レイアウト解析で抽出 |
| 産地・地域             | 生産国・産地名             |
| 焙煎度                 | Light / Medium / Dark 等   |
| 精製方法               | Natural / Washed 等        |
| フレーバーノーツ       | 風味の説明                 |
| 品種・標高・農家・農園 | 詳細スペック               |
| Limited フラグ         | 限定商品かどうか           |

**テイスティングノート:**
各コーヒー豆に対して飲んだ感想・スコアを記録できます。

---

### 3. ストップウォッチ・学習タイマー

勉強・作業時間を計測して記録するストップウォッチです

**できること:**

- スタート / ストップ / リセット / ラップ計測
- カテゴリ入力（Programming / English / Math など自由入力）
- 計測終了後に学習記録として保存
- タブを閉じてもタイマーの状態を保持（`localStorage` によるセッション保持）
- 複数デバイスで同じタイマー状態を共有（5 秒ごとにサーバーへ自動同期）

---

### 4. 学習統計・予測

蓄積した学習記録をもとに、統計データと今日の学習時間の予測を表示します。

**表示される情報:**

| 指標               | 説明                               |
| ------------------ | ---------------------------------- |
| 今週の曜日別グラフ | 日〜土の学習時間を棒グラフで可視化 |
| 週別平均           | 過去の週ごとの平均学習時間         |
| 月別平均           | 過去の月ごとの平均学習時間         |
| 累計学習時間       | 全記録の合計時間                   |
| 今日の予測学習時間 | 同じ曜日の過去記録の平均から算出   |

---

### 5. ボイスメモ

スマートフォンのマイクを使って音声を録音し、テキストのメモ・タグとともに保存できます。

**できること:**

- ブラウザのマイクで音声録音（WebM / M4A / WAV 対応）
- タイトル・メモ・タグの付与
- 録音データをローカル（IndexedDB）に保存
- サーバーへの音声ファイル同期（アップロード・更新・削除）
- オフライン時はローカルに保持し、オンライン復帰時に自動同期
